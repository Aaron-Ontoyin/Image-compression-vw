Compresion Settings Help

Compression Type

Lossless This option style="background-color: aliceblue;" retains the highest quality but may result in larger file sizes. Use it when preserving image quality is crucial.
Lossy Choose this option style="background-color: aliceblue;" to reduce saving storage space.

Compression Magnitude:

1-3 Use for minimal compression and the best quality. Suitable for high-quality images but with larger file sizes.
4-7 Offers a balance between quality and file size. Ideal for most situations.
8-10 Use for strong compression and smaller file sizes. Quality may decrease, so choose this for web or casual use.

Resolution

Full Size Keeps the image at its original resolution.
High Slightly reduces resolution while maintaining good quality.
Medium Moderately reduces resolution for a balance between quality and file size.
Low Significantly reduces resolution to save space but may impact quality.

Meta Data Preservation

Preserve Keeps all image metadata (EXIF, location, etc.).
Strip Removes all metadata to reduce file size. Choose this for privacy or smaller files.

Color Space

RGB Standard color space for most images.
CMYK Use for printing or specialized color needs.
Grayscale Converts the image to black and white.

Optimization Level

Low Minimal optimization for the best quality.
High Good balance between quality and file size.
High Higher optimization, suitable for web or digital sharing.
Maximum Strong optimization for the smallest file size but lower quality.

These settings give you control over your image compression. Select the options that best match your needs, whether it's preserving quality or saving space.
